---
title: "List memberships"
---

Retrieves a list of membership plans for your organisation.

**GET /memberships**

#### **Authorizations**

**Authorization**

**string**

**header**

**required**

Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.

#### **Query Parameters**

**status**

**enum\<string\>**

Filter results to only memberships matching the provided status.

Available options:

`active`,

`inactive`,

`archived`

**limit**

**integer**

**default: 50**

The maximum number of memberships to retrieve.

Required range: `1 <= x <= 250`

**cursor**

**string**

Cursor for the page of results to retrieve. Omit to retrieve the first page.

#### **Response**

**200 - application/json**

**memberships**

**object[]**

Hide child attributes

**memberships.uuid**

**string\<uuid\>**

Unique UUID identifier for the membership.

Example:

`"a1b2c3d4-e5f6-7890-abcd-ef1234567890"`

**memberships.org_uuid**

**string\<uuid\>**

The UUID of the organisation this membership belongs to.

Example:

`"05ca3845-6571-49a5-8d95-2f5d3c13ce5a"`

**memberships.name**

**string**

The display name of the membership plan.

Example:

`"Gold Member"`

**memberships.description**

**string | null**

A description of the membership plan.

Example:

`"Get exclusive discounts and free shipping every month."`

**memberships.identifier**

**string | null**

A unique slug-style identifier for the membership.

Example:

`"GOLDMEMBER"`

**memberships.merchant_code**

**string | null**

A merchant-defined code for this membership plan.

Example:

`"GOLD-2026"`

**memberships.price**

`9.99`

`"percentage"`

**memberships.status**

`active`,
