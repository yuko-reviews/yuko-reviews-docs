---
title: "Checkout points estimate"
---

Estimates how many loyalty points a customer will earn for a given checkout. Designed for third-party checkout integrations (e.g. GoKwik) to show a real-time points preview before the order is placed. Customer is optional — guest checkouts are fully supported.

**POST /public/checkout/points-estimate**

#### **Authorizations**

**Authorization**

**string**

**header**

**required**

Bearer authentication header of the form `Bearer <token>`, where `<token>` is your REST API token.

#### **Request Body**

**customer**

**object | null**

The customer placing the order. Optional — omit entirely for guest checkouts.

Show child attributes

**customer.id**

**string | null**

The platform customer ID. Used to look up the customer if `email` is not provided.

Example:

`"cus_847362"`

**customer.email**

**string\<email\> | null**

The customer's email address. Takes priority over `id` when both are provided.

Example:

`"rahul.sharma@example.com"`

**currency**

**string**

**required**

ISO 4217 currency code for the order.

Required length: `3`

Example:

`"INR"`

**subtotal**

**number**

**required**

The order subtotal (excluding shipping and tax) used as the base for points calculation.

Example:

`2399.00`

**items**

**object[]**

**required**

The line items in the cart. Must contain at least one item.

Show child attributes

**items[].product_id**

**string**

**required**

The platform product ID.

Example:

`"prod_1001"`

**items[].variant_id**

**string**

**required**

The platform variant/SKU ID.

Example:

`"var_1001_red_m"`

**items[].price**

**number**

**required**

The unit price of this item.

Example:

`999.00`

**items[].quantity**

**integer**

**required**

The quantity ordered.

Required range: `1 <= x`

Example:

`1`

**items[].total**

**number**

**required**

The line total (`price × quantity`) used for item-level points calculation.

Example:

`999.00`

#### **Response**

**200 - application/json**

**data**

**object**

Hide child attributes

**data.points_earned**

**integer**

The estimated number of loyalty points the customer will earn after completing the purchase.

Example:

`120`

**data.message**

**string**

A human-readable message to display to the customer at checkout.

Example:

`"You will earn 120 points after completing this purchase"`

<ResponseExample>

```json 200
{
  "data": {
    "points_earned": 120,
    "message": "You will earn 120 points after completing this purchase"
  }
}
```

```json 200 (guest checkout or no matching campaign)
{
  "data": {
    "points_earned": 0,
    "message": "You will earn 0 points after completing this purchase"
  }
}
```

```json 422 (validation error)
{
  "error": {
    "code": "validation_error",
    "message": "The given data was invalid",
    "details": {

  }
```
