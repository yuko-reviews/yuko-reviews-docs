---
title: "List customers"
---

Retrieves a list of customers based on a set of provided filters.

**GET**

/

**customers**

**Try it**

This endpoint requires the **customer:read** scope.

Results are sorted by `id` in descending order. To lookup customers by email, include the `email` query parameter. Note however that Yuko does not enforce uniqueness on email, so multiple customer records may be returned if they have the same email address.

#### **Authorizations**

[**​**](https://dev.smile.io/api/resources/customers/list-customers#authorization-authorization)

**Authorization**

**string**

**header**

**required**

Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.

#### **Query Parameters**

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-email)

**email**

**string**

Filter results to only customers with the provided email address.

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-state)

**state**

**enum\<string\>**

Filter results to only customers with the provided state.

Available options:

`candidate`, 

`member`, 

`disabled` 

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-updated-at-min)

**updated_at_min**

**string\<date-time\>**

Filter results to only customers updated after the provided date and time.

Example:

`"2024-04-04T15:10:42.030Z"`

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-limit)

**limit**

**integer**

**default:50**

The maximum number of customers to retrieve.

Required range: `1 <= x <= 250`

Example:

`1`

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-cursor)

**cursor**

**string**

Cursor for the page of customers to be retrieve.

[**​**](https://dev.smile.io/api/resources/customers/list-customers#parameter-include)

**include**

**enum\<string\>**

A comma-separated list of related objects to include in the response.

Available options:

`vip_status` 

#### **Response**

**200 - application/json**

The customers were successfully retrieved.
