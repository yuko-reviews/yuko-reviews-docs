---
title: "List earning rules"
---

ProductsRetrieves a list of enabled earning rules.

**GET** / **earning_rules - Try it**

This endpoint requires the **earning_rule: read** scope.

Results are sorted by `id` in descending order.

#### **Authorizations:**

\
**Authorization**

**string**

**header**

**required**

Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.

#### **Query Parameters**

[**​**](https://dev.smile.io/api/resources/earning-rules/list-earning-rules#parameter-limit)

**limit**

**integer**

**default:50**

The maximum number of earning rules to retrieve.

Required range: `1 <= x <= 250`

[**​**](https://dev.smile.io/api/resources/earning-rules/list-earning-rules#parameter-cursor)

**cursor**

**string**

Cursor for the page of earning rules to retrieve.
