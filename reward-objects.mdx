---
title: "Reward Object"
---

A reward is the configuration for a type of coupon/discount/benefit/perk that can be issued to a customer. When the configuration is used to issue a reward, a reward fulfillment is generated.

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-id)

**id**

**number**

Unique identifier for the reward.

Example:

`924565472`

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-name)

**name**

**string**

The name of the reward.

Example:

`"Order discount"`

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-description)

**description**

**string**

A description of the reward.

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-image-url)

**image_url**

**string**

An image for the reward.

Example:

`"https://platform-images.smilecdn.co/3755938.png"`

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-created-at)

**created_at**

**string\<date-time\>**

The date and time when the reward was created.

Example:

`"2024-04-04T15:10:42.030Z"`

[**​**](https://dev.smile.io/api/resources/rewards/reward-object#schema-updated-at)

**updated_at**

**string\<date-time\>**

The date and time when the reward was last updated.

Example:

`"2024-04-04T15:10:42.030Z"`
