### **Navigation**

Yuko Dashboard → **Workflows → Review Request Workflow**

![rr.png](/images/rr.png)

### **Purpose**

Automatically send a review request email after a customer’s order is fulfilled.\
This is the most common workflow for collecting verified reviews.

### **How It Works**

**Trigger:** Order Fulfilled\
**Condition:** Customer has not received a review request yet\
**Action:** Send “Review Request Email” after a set delay

### **Setup Steps**

1. Go to **Workflows → Review Request Workflow**.
2. Set your **trigger** to “Order Fulfilled” or “Order Completed.”
3. Choose a **delay** — typically 5 to 7 days after fulfillment.
4. Select your **Review Request Email template.**
5. Save and activate the workflow.
