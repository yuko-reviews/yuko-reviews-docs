---
title: "List points products"
---

Retrieves a list of points products based on the specified filters.

**GET** / **points_products - Try it**

This endpoint requires the **points_product:read** scope.

#### **Authorizations**

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#authorization-authorization)

**Authorization**

**string**

**header**

**required**

Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.

#### **Query Parameters**

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#parameter-exchange-type)

**exchange_type**

**enum\<string\>**

Filter results to only points products with the provided `exchange_type`.

Available options:

`fixed`,

`variable`

Example:

`"variable"`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#parameter-page-size)

**page_size**

**integer**

**default:50**

The maximum number of points products to retrieve.

Required range: `1 <= x <= 250`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#parameter-page)

**page**

**integer**

**default:1**

The page of points products to retrieve.

#### **Response**

**200 - application/json**

The points products were successfully retrieved.

**enum\<string\>**

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-exchange-description)

**points\_**[**products.exchange**](http://products.exchange)**\_description**

**string**

A human readable description of how a customer spends points on this reward. It includes the points branding for the program.

Example:

`"100 Points = $1 off"`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-points-price)

**points_products.points_price**

**integer**

Number of points needed to purchase this reward. Only present when `exchange_type` is `fixed`.

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-variable-points-step-one-of-0)

**points_products.variable_points_step**

**integer | null**

Imagine there's a points slider your customers can drag to spend more or less points on a reward. The `variable_points_step` represents the number of points between each notch on the slider. E.g. If this value is set to 100, it means customers can spend their points in increments of 100. They will drag the slider from 100, 200, 300, etc. Only present when `exchange_type` is `variable`.

Example:

`100`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-variable-points-step-reward-value-one-of-0)

**points_products.variable_points_step_reward_value**

**integer | null**

Imagine there's a points slider your customers can drag to spend more or less points on a reward. The `variable_points_step_reward_value` represents the corresponding reward value for each step increment on the slider. E.g. if this value is set to 1, and the variable_points_step is set to 100, the customer will get \$1 off for every 100 points they spend. Only present when `exchange_type` is `variable.`

Example:

`1`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-variable-points-min-one-of-0)

**points_products.variable_points_min**

**integer | null**

The minimum amount of points the customer must spend to get this reward. Normally the minimum value of the points slider. Only present when `exchange_type` is `variable`.

Example:

`100`

[**​**](https://dev.smile.io/api/resources/points-products/list-points-products#response-points-products-items-variable-points-max-one-of-0)

**points_products.variable_points_max**

**integer | null**

The maximum amount of points the customer must spend to get this reward. Normally the maximum value of the points slider. Only present when `exchange_type` is `variable`.

Example:

`5000`
